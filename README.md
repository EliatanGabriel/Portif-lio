# 🚀 Portfólio - Eliatan Gabriel

Portfólio pessoal desenvolvido com foco em apresentar minha trajetória, habilidades, projetos e experiências na área de tecnologia.

O projeto foi criado com HTML5 e CSS3, priorizando design moderno, responsividade, organização visual e experiência do usuário.

---

## 📌 Sobre o Projeto

Este portfólio foi desenvolvido para centralizar:

- Informações profissionais
- Projetos pessoais
- Competências técnicas
- Contatos profissionais
- Experiências na área de TI

Além disso, o projeto também serve como prática contínua em desenvolvimento web e construção de interfaces modernas.

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git & GitHub

---

## 🎯 Funcionalidades

- Layout moderno e responsivo
- Navbar fixa
- Seção de projetos
- Cards interativos
- Links para GitHub e LinkedIn
- Download de currículo
- Estrutura semântica
- Design adaptado para mobile
